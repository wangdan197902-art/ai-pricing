## 1. 业务定位与价值主张

### 1.1 一句话定位

> "AI工具界的PriceRunner — 一站式对比500+ AI工具的价格、功能与省钱方案，让用户每次订阅都买到最划算的方案。"

### 1.2 核心问题与解决方案

| 用户痛点 | 现状 | 我们方案 |
|---------|------|---------|
| AI工具太多，不知道选哪个 | 信息分散在Reddit、Twitter、厂商官网 | 统一数据库500+工具，按用途/价格筛选 |
| 不知道哪个工具性价比最高 | 厂商自己说好，缺乏中立对比 | 中立对比矩阵，按维度量化评分 |
| 想省钱但不知道有什么折扣 | 学生折扣、年付优惠信息分散 | 每工具专门省钱攻略页 + 订阅计算器 |
| 价格随时变，买完发现降价 | 没有价格监测和提醒 | 每日价格爬取+变更告警+历史价格曲线 |
| 非英语用户难找本地化信息 | 多数评测只有英文 | 10语种全覆盖，本地化CPC更高 |
| 企业采购难决策 | 缺乏批量对比和API | 企业对比报告+API接入 |

### 1.3 目标用户画像

#### 用户类型 A：AI重度个人用户（30%）

- **画像**：25-40岁，技术/创作从业者，月均订阅3-5个AI工具
- **场景**：每季度评估是否切换工具、合并订阅、利用学生折扣
- **核心需求**：省钱攻略、对比矩阵、API vs 订阅计算
- **付费意愿**：高（已经付$50-100/月订阅）
- **典型LTV**：通过联盟转化 $30-80/年

#### 用户类型 B：企业AI采购决策者（25%）

- **画像**：IT经理/CTO/采购经理，决策团队5-50人AI工具预算
- **场景**：选型阶段，对比5-10个工具，需要ROI测算
- **核心需求**：团队价格、企业计划、API定价、合规性
- **付费意愿**：极高（B2B高客单价）
- **典型LTV**：企业咨询 $500-5000/单

#### 用户类型 C：内容创作者（20%）

- **画像**：YouTuber/Blogger/设计师，依赖AI做内容生产
- **场景**：寻找Midjourney/Runway/Suno的最佳替代品
- **核心需求**：按创作类型筛选、免费试用、教程链接
- **付费意愿**：中
- **典型LTV**：联盟转化 $20-50/年

#### 用户类型 D：开发者（15%）

- **画像**：全栈/AI工程师，调用OpenAI/Claude/Gemini API
- **场景**：API定价对比、token成本估算、模型性能对比
- **核心需求**：API定价表、token计算器、模型benchmark
- **付费意愿**：高
- **典型LTV**：API联盟 $50-200/年

#### 用户类型 E：学生/教育用户（10%）

- **画像**：大学生/研究生，预算敏感
- **场景**：寻找学生折扣、免费替代方案
- **核心需求**：学生折扣攻略、免费工具清单、教育优惠
- **付费意愿**：低但流量大
- **典型LTV**：广告 $5-15/年

### 3.2 核心功能详述

#### 3.2.1 工具查询（Tool Browse）

**功能描述**：浏览500+ AI工具，按分类/厂商/价格/用途筛选。

**关键页面**：
- `/en/tools/` — 全部工具列表，支持分页+筛选
- `/en/tools/?category=writing` — 按分类筛选
- `/en/tools/?price=free` — 按价格筛选
- `/en/tools/?usecase=coding` — 按用途筛选

**功能点**：
- 12个一级分类（chat/writing/image/video/audio/code/data/search/agent/design/productivity/other）
- 多维筛选：分类、价格区间、厂商、用途、评分
- 排序：人气、价格、评分、最新
- 每工具卡片：名称、Logo、起步价、核心功能、评分、查看按钮

**SEO 关键词**："AI writing tools", "AI image generators", "best AI chatbots 2026"

#### 3.2.2 价格对比（Comparison）

**功能描述**：2-5个工具并排对比，10维度量化。

**关键页面**：
- `/en/comparison/chatgpt-vs-claude-pricing/` — 预设对比页
- `/en/comparison/?tools=chatgpt-plus,claude-pro,gemini-advanced` — 动态对比

**对比维度**：
1. 起步价格（月/年）
2. 上下文窗口
3. 模型质量（基于公开benchmark）
4. 编码能力
5. 写作能力
6. 研究能力
7. API定价
8. 免费层
9. 多模态支持
10. 团队/企业计划

**功能点**：
- 视觉化矩阵（✓/✗/数值）
- 高亮优劣势
- 推荐使用场景（"适合写作", "适合编程", "适合研究"）

- 联盟CTA按钮

**SEO关键词**："ChatGPT vs Claude", "AI tools comparison 2026", "best AI for coding"

#### 3.2.4 订阅计算器（Subscription Calculator）

**功能描述**：根据用户使用量，推荐最划算方案。

**关键页面**：`/en/calculator/`

**输入**：
- 主要用途（聊天/写作/编码/图像/视频）
- 月均消息量（<100/100-1000/1000-10000/>10000）
- 是否需要API
- 团队人数
- 预算上限

**输出**：
- 推荐方案（1-3个）
- 月度/年度成本估算
- API vs 订阅对比
- 节省金额对比

**实现**：纯前端JS，数据从`data/tools/*.json`读取。

```javascript
// assets/js/calculator.js 节选
function recommend(usecase, monthlyMsgs, needsAPI, teamSize, budget) {
  const candidates = window.TOOLS_DATA.filter(t => 
    t.usecases.includes(usecase) && 
    t.pricing.plans.some(p => p.price * (teamSize || 1) <= budget)
  );
  // 计算每候选的每消息成本
  candidates.forEach(t => {
    const plus = t.pricing.plans.find(p => p.name === 'Plus') || t.pricing.plans[1];
    t.cost_per_msg = plus ? plus.price / monthlyMsgs : Infinity;
    t.api_cost = needsAPI ? t.pricing.price_per_query * monthlyMsgs : 0;
    t.total_monthly = (plus?.price || 0) + t.api_cost;
  });
  return candidates.sort((a,b) => a.total_monthly - b.total_monthly).slice(0, 3);
}
```

#### 3.2.5 价格告警（Price Alert）

**功能描述**：用户订阅某工具价格变化通知。

**实现路径**：
- 短期：邮件订阅(Buttondown)，每日扫描价格变更批量推送
- 中期：Web Push(OneSignal免费)，实时推送
- 长期：APP(暂时不开发)

**告警触发**：
- 价格上涨 ≥5%
- 价格下降 ≥5%
- 新增计划
- 计划下架

#### 3.2.6 API（企业）

**功能描述**：付费API供企业集成价格数据。

**端点**：
- `GET /api/v1/tools` — 全部工具
- `GET /api/v1/tools/{slug}` — 单工具详情
- `GET /api/v1/tools/{slug}/prices` — 价格历史
- `GET /api/v1/comparisons` — 对比页

**实现**：Cloudflare Workers + KV，免费层100K请求/天，足够MVP。

#### 3.2.7 多语种（i18n）

**功能描述**：10语种独立URL，每页面×10镜像。

**URL结构**：
- `/en/tool/chatgpt-plus/pricing/` — 英文
- `/zh/tool/chatgpt-plus/pricing/` — 中文
- `/ja/tool/chatgpt-plus/pricing/` — 日文
- ...（10语种）

**翻译策略**：英文为源，GPT-4o翻译到其他9语种，Claude审核。

- **DMCA**：提供DMCA联系方式
