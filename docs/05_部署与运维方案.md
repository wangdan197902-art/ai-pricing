- SSL/TLS 模式：**Full (strict)**

- 启用 TLS 1.3
- 启用 Automatic HTTPS Rewrites
- 启用 Always Use HTTPS
- 启用 HSTS：max-age=1 year, includeSubDomains, preload
- 启用 OCSP Stapling

## 4. GitHub Actions CI/CD

### 4.1 deploy.yml（完整版）

`.github/workflows/deploy.yml`：

```yaml
name: Build & Deploy

on:
  push:
    branches: [main]
  workflow_dispatch:

permissions:
  contents: read
  deployments: write

concurrency:
  group: deploy-${{ github.ref }}
  cancel-in-progress: false

jobs:
  build-deploy:
    runs-on: ubuntu-latest
    timeout-minutes: 15
    steps:
      - name: Checkout
        uses: actions/checkout@v4
        with:
          fetch-depth: 1

      - name: Setup Hugo
        uses: peaceiris/actions-hugo@v3
        with:
          hugo-version: '0.140.0'
          extended: true

      - name: Setup Node
        uses: actions/setup-node@v4
        with:
          node-version: '20'

      - name: Setup Python
        uses: actions/setup-python@v5
        with:
          python-version: '3.11'

      - name: Cache Hugo resources
        uses: actions/cache@v4
        with:
          path: |
            resources/_gen
            .hugo_build.lock
          key: hugo-${{ runner.os }}-${{ hashFiles('config/**', 'content/**', 'data/**', 'layouts/**', 'assets/**') }}
          restore-keys: |
            hugo-${{ runner.os }}-

      - name: Install Python deps
        run: pip install -q jsonschema requests

      - name: Validate data
        run: python3 scripts/validate_project.py .

      - name: Build Hugo (production)
        env:
          HUGO_ENV: production
        run: |
          hugo \
            --minify \
            --gc \
            --enableGitInfo \
            --baseURL "https://aitoolspricing.com/" \
            --destination public

      - name: Generate llms.txt
        run: python3 scripts/generate_llms_txt.py --out public/llms.txt

      - name: Generate llms-full.txt
        run: python3 scripts/generate_llms_full.py --out public/llms-full.txt

      - name: Test internal links
        run: |
          pip install -q linkcheck
          python3 scripts/test_links.py public/ || true

      - name: Lighthouse CI
        uses: treosh/lighthouse-ci-action@v11
        with:
          urls: |
            https://aitoolspricing.com/en/
            https://aitoolspricing.com/en/tool/chatgpt-plus/pricing/
          budgetPath: .lighthouse-budget.json
          temporaryPublicStorage: true
        continue-on-error: true

      - name: Upload build artifact
        uses: actions/upload-artifact@v4
        with:
          name: public
          path: public/
          retention-days: 7

      - name: Deploy to Cloudflare Pages
        uses: cloudflare/pages-action@v1
        with:
          apiToken: ${{ secrets.CLOUDFLARE_API_TOKEN }}
          accountId: ${{ secrets.CLOUDFLARE_ACCOUNT_ID }}
          projectName: aitoolspricing
          directory: public
          gitHubComments: true

      - name: Purge Cloudflare cache
        env:
          CF_ZONE: ${{ secrets.CLOUDFLARE_ZONE_ID }}
          CF_TOKEN: ${{ secrets.CLOUDFLARE_API_TOKEN }}
        run: |
          curl -X POST "https://api.cloudflare.com/client/v4/zones/${CF_ZONE}/purge_cache" \
            -H "Authorization: Bearer ${CF_TOKEN}" \
            -H "Content-Type: application/json" \
            --data '{"purge_everything": true}'

      - name: Notify Google Indexing
        env:
          GOOGLE_INDEXING_CREDENTIALS: ${{ secrets.GOOGLE_INDEXING_CREDENTIALS }}
        run: |
          if [ -n "$GOOGLE_INDEXING_CREDENTIALS" ]; then
            echo "$GOOGLE_INDEXING_CREDENTIALS" > /tmp/google-creds.json
            python3 scripts/notify_indexing.py \
              --urls "https://aitoolspricing.com/en/,https://aitoolspricing.com/sitemap.xml" \
              --credentials /tmp/google-creds.json
          fi

      - name: Notify on success
        if: success()
        run: python3 scripts/deploy_notify.py --status success --commit ${{ github.sha }}

      - name: Notify on failure
        if: failure()
        run: python3 scripts/deploy_notify.py --status failure --commit ${{ github.sha }}
```

### 4.2 daily-prices.yml（完整版）

`.github/workflows/daily-prices.yml`：

```yaml
name: Daily Price Monitor

on:
  schedule:
    - cron: '0 6 * * *'  # UTC 06:00 = Beijing 14:00
  workflow_dispatch:

permissions:
  contents: write
  pull-requests: write

jobs:
  check-prices:
    runs-on: ubuntu-latest
    timeout-minutes: 30
    steps:
      - uses: actions/checkout@v4
        with:
          fetch-depth: 0

      - uses: actions/setup-python@v5
        with:
          python-version: '3.11'

      - name: Configure git
        run: |
          git config --global user.name "AI Pricing Bot"
          git config --global user.email "bot@aitoolspricing.com"

      - name: Install deps
        run: pip install -q requests beautifulsoup4 lxml jsonschema python-dateutil

      - name: Fetch latest prices
        env:
          ANTHROPIC_API_KEY: ${{ secrets.ANTHROPIC_API_KEY }}
          OPENAI_API_KEY: ${{ secrets.OPENAI_API_KEY }}
          SLACK_WEBHOOK: ${{ secrets.SLACK_WEBHOOK }}
        run: python3 scripts/fetch_prices.py --with-diff --alert

      - name: Detect changes
        id: diff
        run: |
          if [[ -n "$(git status --porcelain data/prices/ data/tools/)" ]]; then
            echo "changed=true" >> $GITHUB_OUTPUT
            git diff --stat data/prices/ data/tools/ > /tmp/diff.txt
            {
              echo "## Price changes detected"
              echo ""
              echo '```'
              cat /tmp/diff.txt
              echo '```'
              echo ""
              echo "Generated by daily-prices.yml at $(date -u)"
            } > /tmp/pr.md
          else
            echo "changed=false" >> $GITHUB_OUTPUT
            echo "No price changes detected." > /tmp/pr.md
          fi

      - name: Trigger content regen for changed tools
        if: steps.diff.outputs.changed == 'true'
        env:
          ANTHROPIC_API_KEY: ${{ secrets.ANTHROPIC_API_KEY }}
          OPENAI_API_KEY: ${{ secrets.OPENAI_API_KEY }}
          GEMINI_API_KEY: ${{ secrets.GEMINI_API_KEY }}
        run: |
          CHANGED=$(git diff --name-only HEAD data/prices/ | sed 's|data/prices/||; s|.json||' | tr '\n' ' ')
          if [ -n "$CHANGED" ]; then
            echo "Regenerating content for: $CHANGED"
            python3 scripts/generate_comparison.py --tools "$CHANGED" || true
            python3 scripts/generate_guide.py --tool $(echo $CHANGED | tr ' ' '\n' | head -1) || true
            python3 scripts/translate_content.py --tool $(echo $CHANGED | tr ' ' '\n' | head -1) --target-langs zh,ja,ko,es,fr,de,pt,ru,ar || true
          fi

      - name: Validate
        if: steps.diff.outputs.changed == 'true'
        run: python3 scripts/validate_project.py .

      - name: Audit content
        if: steps.diff.outputs.changed == 'true'
        env:
          ANTHROPIC_API_KEY: ${{ secrets.ANTHROPIC_API_KEY }}
        run: python3 scripts/audit_content.py --all || true

      - name: Create Pull Request
        if: steps.diff.outputs.changed == 'true'
        uses: peter-evans/create-pull-request@v6
        with:
          token: ${{ secrets.GITHUB_TOKEN }}
          commit-message: "chore(prices): daily price update $(date -u +%Y-%m-%d)"
          branch: daily-prices-$(date -u +%Y%m%d)
          delete-branch: true
          title: "Daily price update $(date -u +%Y-%m-%d)"
          body-path: /tmp/pr.md
          labels: |
            prices
            automated
          reviewers: wangdan

      - name: Send Slack alert
        if: steps.diff.outputs.changed == 'true'
        env:
          SLACK_WEBHOOK: ${{ secrets.SLACK_WEBHOOK }}
        run: python3 scripts/deploy_notify.py --event price-change

      - name: Auto-merge PR if no audit failures
        if: steps.diff.outputs.changed == 'true'
        env:
          GH_TOKEN: ${{ secrets.GITHUB_TOKEN }}
        run: |
          sleep 30  # wait for PR creation
          PR_NUMBER=$(gh pr list --head daily-prices-$(date -u +%Y%m%d) --json number -q '.[0].number')
          if [ -n "$PR_NUMBER" ]; then
            # Check if audit passed
            if python3 scripts/check_audit_pass.py; then
              gh pr merge $PR_NUMBER --squash --auto --delete-branch
              echo "PR $PR_NUMBER will auto-merge after checks"
            fi
          fi
```

### 4.3 weekly-content.yml（完整版）

`.github/workflows/weekly-content.yml`：

```yaml
name: Weekly Content Production

on:
  schedule:
    - cron: '0 3 * * 1'  # 每周一 UTC 03:00
  workflow_dispatch:

permissions:
  contents: write
  pull-requests: write

jobs:
  produce:
    runs-on: ubuntu-latest
    timeout-minutes: 60
    steps:
      - uses: actions/checkout@v4
        with:
          fetch-depth: 0

      - uses: actions/setup-python@v5
        with:
          python-version: '3.11'

      - name: Configure git
        run: |
          git config --global user.name "AI Pricing Bot"
          git config --global user.email "bot@aitoolspricing.com"

      - name: Install deps
        run: pip install -q requests beautifulsoup4 lxml jsonschema python-dateutil Pillow

      - name: Discover new tools
        env:
          OPENAI_API_KEY: ${{ secrets.OPENAI_API_KEY }}
        run: python3 scripts/fetch_tools.py --limit 20 --new-only

      - name: Generate comparisons for new tools
        env:
          ANTHROPIC_API_KEY: ${{ secrets.ANTHROPIC_API_KEY }}
        run: |
          NEW_TOOLS=$(git diff --name-only HEAD data/tools/ | sed 's|data/tools/||; s|.json||' | tr '\n' ',')
          if [ -n "$NEW_TOOLS" ]; then
            python3 scripts/generate_comparison.py --tools "$NEW_TOOLS"
          fi

      - name: Generate guides for new tools
        env:
          ANTHROPIC_API_KEY: ${{ secrets.ANTHROPIC_API_KEY }}
        run: |
          for slug in $(git diff --name-only HEAD data/tools/ | sed 's|data/tools/||; s|.json||'); do
            python3 scripts/generate_guide.py --tool $slug || true
          done

      - name: Translate to 9 languages
        env:
          OPENAI_API_KEY: ${{ secrets.OPENAI_API_KEY }}
        run: |
          for slug in $(git diff --name-only HEAD data/tools/ | sed 's|data/tools/||; s|.json||'); do
            python3 scripts/translate_content.py --tool $slug --target-langs zh,ja,ko,es,fr,de,pt,ru,ar || true
          done

      - name: Generate images
        env:
          GEMINI_API_KEY: ${{ secrets.GEMINI_API_KEY }}
        run: python3 scripts/generate_image.py --new-only || true

      - name: Audit content
        env:
          ANTHROPIC_API_KEY: ${{ secrets.ANTHROPIC_API_KEY }}
        run: python3 scripts/audit_content.py --new-only || true

      - name: Validate
        run: python3 scripts/validate_project.py .

      - name: Create PR
        uses: peter-evans/create-pull-request@v6
        with:
          token: ${{ secrets.GITHUB_TOKEN }}
          commit-message: "feat(content): weekly batch $(date -u +%Y-%m-%d)"
          branch: weekly-content-$(date -u +%Y%m%d)
          delete-branch: true
          title: "Weekly content batch $(date -u +%Y-%m-%d)"
          body: |
            ## Weekly Content Production

            This PR contains:
            - New tools discovered this week
            - Generated comparison pages
            - Generated saving guides
            - 9-language translations
            - Generated images
            - AI audit report

            Generated by weekly-content.yml at $(date -u)
          labels: |
            content
            automated
          reviewers: wangdan
```

### 6.2 关键数据备份

**自动备份到另一个GitHub仓库**（防止单点丢失）：

`.github/workflows/backup.yml`：

```yaml
name: Daily Backup

on:
  schedule:
    - cron: '0 1 * * *'  # 每日 UTC 01:00
  workflow_dispatch:

jobs:
  backup:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
        with:
          fetch-depth: 0

      - name: Mirror to backup repo
        env:
          BACKUP_TOKEN: ${{ secrets.BACKUP_GITHUB_TOKEN }}
        run: |
          git remote add backup https://x-access-token:${BACKUP_TOKEN}@github.com/aitoolspricing/ai-price-compare-backup.git
          git push --force backup main
          echo "Backup completed at $(date -u)"
```

### 6.3 恢复流程

**Scenario 1: GitHub仓库误删**
1. 从备份仓库 `ai-price-compare-backup` 恢复
2. 或从Cloudflare Pages最后一次构建产物反向恢复

**Scenario 2: 内容误删**
1. `git log --diff-filter=D --summary` 找到删除commit
2. `git checkout <commit>^ -- path/to/file` 恢复
3. 提交+部署

**Scenario 3: 价格数据损坏**
1. `git log data/prices/{slug}.json` 找历史
2. 恢复到上次有效版本
3. 重新跑 fetch_prices.py

**Scenario 4: Cloudflare Pages项目损坏**
1. 重新创建Pages项目
2. 重新绑定GitHub仓库
3. 重新配置环境变量
4. 触发部署

## 11. 故障处理手册

### 11.1 故障分级

| 级别 | 定义 | 响应SLA | 例子 |
|------|------|---------|------|
| P0 | 全站不可用 | 15min | Cloudflare宕机 |
| P1 | 关键功能不可用 | 1h | 部署失败 |
| P2 | 部分功能不可用 | 4h | 某工具页404 |
| P3 | 体验问题 | 24h | Lighthouse分数下降 |

### 11.2 故障处理流程

```
告警触发
    ↓
确认级别(P0-P3)
    ↓
通知相关人(P0/P1: 创始人)
    ↓
排查(根据Scenario)
    ↓
修复
    ↓
验证
    ↓
事后总结(写入 /docs/postmortem/)
```

### 11.3 常见故障Scenario

#### Scenario A: 站点不可用 (P0)

**症状**：UptimeRobot 5min连续2次失败
**排查**：
1. 检查 Cloudflare Status (https://www.cloudflarestatus.com/)
2. 检查 GitHub Actions 最近部署是否成功
3. 检查域名 DNS 是否正确
4. 用 curl 测试 TTFB
5. 检查 Cloudflare Pages 项目状态

**修复**：
- Cloudflare全挂：等待恢复，启用Vercel备用
- Pages项目损坏：重新创建+部署
- DNS错误：重新配置
- 代码bug：回滚到上次成功部署

#### Scenario B: 部署失败 (P1)

**症状**：deploy.yml 失败
**排查**：
1. 查看 GitHub Actions 日志
2. 常见原因：
   - Hugo构建错误（模板语法）
   - Schema校验失败
   - 链接测试失败
   - Cloudflare API 错误

**修复**：
- 模板错误：修复模板代码
- Schema失败：修复data/*.json
- Cloudflare API：重试或检查token
- 紧急情况：`git revert` + 重新部署

#### Scenario C: 价格采集失败 (P1)

**症状**：daily-prices.yml 失败
**排查**：
1. 查看 fetch_prices.py 输出
2. 检查目标网站是否可达
3. 检查 Claude API 是否限流
4. 检查网络连接

**修复**：
- 厂商网站改版：更新解析逻辑
- Claude限流：等待+重试
- 网络：重试

#### Scenario D: AI API限流 (P2)

**症状**：429 Too Many Requests
**排查**：
1. 查看 API dashboard
2. 检查 quota 使用

**修复**：
- 升级 quota
- 切换到备用模型
- 暂停非关键任务

#### Scenario E: AdSense警告 (P1)

**症状**：收到 AdSense 政策违规邮件
**排查**：
1. 阅读邮件了解具体违规
2. 定位违规页面

**修复**：
- 立即下架违规内容
- 提交整改申诉
- 申诉期间用Ezoic-only
